# Face Database Directory

This directory should contain subdirectories for each person, with images of their faces.

## Structure

```
data/faces/
├── person1/
│   ├── image1.jpg
│   └── image2.jpg
└── person2/
    └── image1.jpg
```

## Instructions

1. Create a subdirectory for each person you want to recognize
2. Add one or more photos of each person to their directory
3. Supported formats: .jpg, .jpeg, .png, .bmp, .gif
4. Tips for best results:
   - Use clear, well-lit photos
   - Ensure the face is clearly visible
   - Multiple photos per person improves accuracy
   - One face per image works best
